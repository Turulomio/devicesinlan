DEVICESINLAN

 

NAME

devicesinlan: Escanéa todos los dispositivos en tu red. Luego permite establecer un alias a tus dispositivos conocidos para que se pueda detectar dispositivos estraños en tu red.  

SYNOPSIS

devicesinlan [-h] [--version] [--debug DEBUG] [ --wizard | --interface | --add | --remove | --list | --load | --save | --reset ]  

DESCRIPCIÓN DEL MODO INTERFAZ GRÁFICO

If you launch devicesinlan without parameters and you are in a graphic system (Linux or Windows) it launches the program with a Qt Interface.

In the app menu you have the followings features:

Dispositivos -> Nuevo escaneado

Dispositivos > Mostrar base de datos

Shows all known devices in a new tab.

Right click allows you to edit known devices database.

Dispositivos -> Carga una lista de dispositivos

Loads a list of known devices in xml format.

Dispositivos -> Guarda la lista de dispositivos

Saves the known devices list to a xml file.

Dispositivos > Borrar base de datos

Removes all known devices.

Esta opción borra todos los dispositivos conocidos de la base de datos.

Configuración > Preferencias

En este dialogo, puedes seleccionar tu idioma favorita y puedes configurar el número de peticiones concurrentes.

Ayuda > Ayuda

Muestra esta ayuda.

Ayuda > Acerca de

Muestra información sobre la licencia y los autores de DevicesInLan.

Ayuda -> Buscar actualizaciones

Checks for updates in DevicesInLan repository.

Ayuda > Salir

Sale del programa.
 

DESCRIPCIÓN DEL MODO CONSOLA

If you launch deviceslan with [ --wizard | --interface | --add | --remove | --list | --load | --save | --reset ], it will be executed in console mode.

--wizard

Lista todos los interfaces de red en el sistema y te permite seleccionar uno y establecer el número de peticiones concurrente. Después de eso, escanéa la red y visualiza una lista de los dispositivos detectados.

--interface

Scans the net of the interface parameter and prints a list of the detected devices.

If a device is not known, it will be showed in red. Devices in green are trusted devices.

--add

Permite añadir un dispositivo conocido desde consola.

--remove

Permite borrar un dispositivo conocido desde consola.

--list

Shows all known devices in database from console.

--load

Loads a list of known devices in xml format.

--save

Saves the known devices list to a xml file.

--debug

Gives debugging information when running DevicesInLAN. It's deactivated by default

The parameter can take this options: CRITICAL, ERROR, WARNING, INFO, DEBUG.

--reset

Removes all known devices.

DEVICESINLAN

 

NAME

devicesinlan: Escanéa todos los dispositivos en tu red. Luego permite establecer un alias a tus dispositivos conocidos para que se pueda detectar dispositivos estraños en tu red.  

SYNOPSIS

devicesinlan [-h] [--version] [--debug DEBUG] [ --wizard | --interface | --add | --remove | --list | --load | --save | --reset ]  

DESCRIPCIÓN DEL MODO INTERFAZ GRÁFICO

Si ejecutas DevicesInLan sin parámetros y tu estás en un sistema gráfico (Linux o Windows), se ejecutará con un interfaz Qt.

En el menú del aplicación tienes las siguientes funcionalidades:

Dispositivos -> Nuevo escaneado

Busca todos los dispositivos existentes en la red y los muestra en una nueva pestaña. Si algún dispositivo no está en la lista de dispositivos conocidos, será mostrado con un fondo rojo. Los dispositivos con un fondo verde son dispositivos confiables

Dispositivos > Mostrar base de datos

Muestra todos los dispositivos conocidos en una nueva pestaña.

Al hacer un click derecho, podrás editar la base de datos de dispositivos conocidos.

Dispositivos -> Carga una lista de dispositivos

Carga una lista de dispositivos conocidos en formato XML.

Dispositivos -> Guarda la lista de dispositivos

Guarda la lista de dispositivos conocidos a un fichero XML.

Dispositivos > Borrar base de datos

Borra todos los dispositivos conocidos.

Esta opción borra todos los dispositivos conocidos de la base de datos.

Configuración > Preferencias

En este dialogo, puedes seleccionar tu idioma favorita y puedes configurar el número de peticiones concurrentes.

Ayuda > Ayuda

Muestra esta ayuda.

Ayuda > Acerca de

Muestra información sobre la licencia y los autores de DevicesInLan.

Ayuda -> Buscar actualizaciones

Busca actualizaciónes en el repositorio de DevicesInLan.

Ayuda > Salir

Sale del programa.
 

DESCRIPCIÓN DEL MODO CONSOLA

Si lanzas devicesinlan con [ --wizard | --interface | --add | --remove | --list | --load | --save | --reset ], será ejecutado en modo consola.

--wizard

Lista todos los interfaces de red en el sistema y te permite seleccionar uno y establecer el número de peticiones concurrente. Después de eso, escanéa la red y visualiza una lista de los dispositivos detectados.

--interface

Escanéa la red del interfaz pasado como parámetro y lista los dispositivos detectados.

Si un dispositivo es desconocido, será mostrado en rojo. Los dispositivos en verde son dispositivos confiables.

--add

Permite añadir un dispositivo conocido desde consola.

--remove

Permite borrar un dispositivo conocido desde consola.

--list

Muestra todos los dispositivos conocidos en la base de datos, desde consola.

--load

Carga una lista de dispositivos conocidos en formato XML.

--save

Guarda la lista de dispositivos conocidos a un fichero XML.

--debug

Muestra información de depuración cuando se ejecuta DevicesInLan. Está desactivada por defecto

El parámetro puede tomar los siguientes valores: CRITICAL, ERROR, WARNING, INFO, DEBUG.

--reset

Borra todos los dispositivos conocidos.
